# Hub de Migração SDM

Dashboard em tempo real para acompanhamento de migrações de clientes.

## 📋 Arquivos

- `index.html` - Aplicação principal (dashboard + login + admin panel)
- `logo-dark.png` - Logo para tema escuro
- `logo-light.png` - Logo para tema claro
- `vercel.json` - Configuração do Vercel

## 🚀 Deploy

1. Faça push para GitHub
2. Conecte ao Vercel
3. Deploy automático
