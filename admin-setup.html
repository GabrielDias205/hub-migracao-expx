⚙️ Setup Inicial

Este script cria o usuário admin no Supabase.

ℹ️ Como funciona:
1. Cole o Service Role Key acima
2. Clique no botão para criar o usuário admin
3. Este script só pode ser rodado uma vez
4. Após criar, acesse o Hub e faça login como admin
5. Lá você poderá criar e gerenciar outros usuários
